# zk-json-cli

## Install
```shell
$ npm install -g zk-json-cli
```

## Use
```shell
$ zk-json-cli --help

zk-json-cli <cmd> [args]

Commands:
  zk-json-cli export [path]  export zookeeper path to json
  zk-json-cli import [path]  import json into zookeeper

Options:
  --version  Show version number                                       [boolean]
  --help     Show help                                                 [boolean]
```
