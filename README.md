# zk-json-cli

## Install
```shell
# npm install -g zk-json-cli
```

## Use
```shell
$ zk-json-cli --help
zk-json-cli <cmd> [args]

Commands:
  zk-json-cli export [path]  export from path
  zk-json-cli import [path]  import from path

Options:
  --version  Show version number                                       [boolean]
  --help     Show help                                                 [boolean]```
